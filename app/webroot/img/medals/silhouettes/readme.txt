Installation:
-------------
Make a backup of "./images/medals" folder then copy the contents 
of this folder into "./images/medals" folder. Replace all files 
when asked.

If you have different game servers and want to use this set for 
a specific game only, create a new folder inside 
"./images/medals" folder with the game's short name as listed
below (i.e. ./images/medals/codwaw/) and copy the contents of
this folder there. XLRstats will automatically detect the new
medals for the game.

  - urt    : Urban Terror
  - cod1   : Call of Duty
  - coduo  : Call of Duty United Offensive
  - cod2   : Call of Duty 2
  - cod4   : Call of Duty 4 Modern Warfare
  - codwaw : Call of Duty World at War
  - wop    : World of Padman
